Projects

Here are some of my recent projects:


Curstomer Churn Prediction Model

This project includes data visualization with mathplotlib, seaborn and plotly. An Exploratory Data Analysis (EDA) and ML models training and testing for the classification task. The best model (LGBM) reached a ROC-AUC metric of 0.9152.

Next Hour Taxi Orders Prediction Model

The data in this project was analyzed using Time Series Analysis (TSA) tools and lag features engineering to develop and test a machinge learning model for the regression task. The best model (CatBoost) reached a RMSE metric of 41.43 which is below the minimum expected value of 48.

Convolutional neural network people's age detector

Using a ResNet50 architecture for Convolutional Neural Networks (CNN), this project aimed to predict the age of a person using a face centered photo. The model reached a Mean Absolute Error (MAE) of 5.48 which is a very good score for this type of projects.
